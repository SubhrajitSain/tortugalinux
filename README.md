## Features

- Comes with most of Fedora's default packages and applications
- Based on Fedora Linux 44 with all of Fedora's latest features
- Agentic capabilities such as Turtagent and Magic Plasmoid
- Connects with all your LibreOffice applications agentically
- Access applications on your Android device seamlessly
- Shake your cursor to open up the Turtagent overlay
- Fully local agentic AI

## Development

TurtLinux is currently in development. We welcome forks and contributions to any part of the operating system including Turtagent. Feel free to make a pull request and we will review it as soon as we can.

### Repository Information

- The ```apps/``` directory contains all the custom pre-installed apps including Turtagent and Magic Plasmoid.
- The ```os/``` direcory contains the files for building the operating system such as the Fedora Kickstart files and shell scripts.
